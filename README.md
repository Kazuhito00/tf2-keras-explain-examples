# tf2-keras-explain-examples
tf2-keras-explain-examplesはGrad-CAM等が簡単に扱えるライブラリtf-explainのJupyter上での実行例です。

# Requirement
 
* Tensorflow 2.0.0(or later)
* OpenCV 3.4.2(or later)
* tf-explain 0.2.1

# Usage
Jupyter上でtf2-keras-explain-examples.ipynbを開いて実行してください。

また、実行例を見るだけであれば、Github上でtf2-keras-explain-examples.ipynbを開くと以下のように見ることが出来ます。
